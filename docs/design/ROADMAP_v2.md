# SpectraQuant-Next v2 Roadmap

## Intraday expansion
- Multi-interval support (1m/5m/15m with shared features).
- Enhanced model persistence and calibration monitoring.

## Alternative data
- Fundamentals, macro, and cross-asset signals.
- Structured data ingestion with schema enforcement.

## Execution simulation
- Slippage and transaction cost models.
- Snapshot-to-trade simulation (analysis-only).

## Research vs production separation
- Clear core vs experimental modules.
- CI enforcement of invariants and artifact validation.
